ngrok 简易使用指南

  • 官网

步骤

1.下载

  • 下载上述网址中的对应操作系统客户端

2.安装

  • 直接解压

3.运行

  • 运行你要映射的本地应用，假设本地应用端口为8080
  • 启动*ngrok*并映射，ngrok http 8080
  • 打开本地浏览器，输入http://localhost:4040，查看映射的公网地址，点击访问即可。

如此就可以愉快的本地调试各种支付回调了
